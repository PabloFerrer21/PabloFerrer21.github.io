DOM STYLE

Representa una declaración de estilo individual
Se puede acceder al objet STYLE desde la seccion del encabezado del documento o desde elementos HTML especificos.

  • alignItems: Establece o devuelve el alineamiento para los items dentro de un contenedor felxible.
  • backgroundColor: Establece el color de fondo de un elemento
  • Color: Establece el color de la letra del texto de un elemento
  • fontFamily: Establece la fuente del texto
  • textAlign: Establece el alineamiento horizontal para el texto

DOM EVENTS

Los eventos HTML DOM permiten que JavaScript registre diferentes controladores de eventos en elementos de un documento HTML
Los eventos se usan normalmente en combinación con funciones y la función no se ejecutará antes de que ocurra el evento.

  • Click: El evento se ejecutara cuando el usuario de click a un elemento
  • DobleClick: El evento se ejecutara cuando el usuario de dobleclick en un elemento
  • KeyDown: El evento se ejecutara cuando el usuario apriete un tecla
  • Keypress: El evento se ejecutara cuando el usuario mantenga pulsada una tecla

DOM ELEMENT

Representa cada uno de la etiquetas XHTML. Se trata del único nodo que puede controlar atributos y el único del que pueden derivar otros nodos

  • Click(): Simula el click del ratón en el elemento
  • Closest(): Busca poer el arbol DOM el elemento mas cercano que concuerde con el CSS selector especificos
  • Contains(): Devuelve true si el nodo desciende de otro nodo, sino devuelve false
  • id: Establece o devuelve el valor del atributo id de un elemento
  • innerHTML: Establece o devuelve el contenido de un elemento
  • querySelector(): Devuelve el primer hijo de un elemento que concuerde con el querySelector CSS especifico de un elemento

DOM DOCUMENT

Nodo raíz del que derivan todos los demás nodo del arbol DOM.
Se puede acceder mediante --> window.document o document

  • addEventListener(): Adjunta un evento en el documento
  • doctype: Devuelve el document type declaration associado con el documento
  • forms: Devuelve la colección de todo elemento form en el documento
  • getElementById: Devuelve el elemento que tiene de ID un atributo con un valor especifico
  • referer: Devuelve el URL del documento que se a cargado en el documento.

PROBANDO DOM
querySelector() y querySelectorAll()

querySelector, tras especificar un selector nos devuelve el primer elemeno resultante, en cambio si la funcion termina con all ( querySelectorAll), nos devuelve un Array con los elementos resultantes.
QuerySelector() : Esta función actúa de un manera similar a un getElementById, ya que podemos seleccionar varios elementos con una ID pero solamente seleccionara el primer elemento. Esta funcion se puede usara para hacer una instancia de document en el qual hacemos una búsqueda sobre cualquier elemento de la página, per a la vez tambien se puede usar para hacer una instancia de un elemento del DOM, como por ejemplo document.body o cualquier elemento que haya sido seleccionado con getElementById().
QuerySelectoAll() : Esta función es similar a un getElementsByTagName, ya que devuelve un conjunto de elementos. QuerySelectorAll(), tiene un ventaja la qual nos permite conseguir todos los elementos de una o varias consultas, simplemente separando con comas, como se hace en CSS. QuerySelectorAll() nos devolverá un Arrat con los elementos y ya podemos manipular el DOM a nuestro gusto.
En resumen estas dos funciones tienen una gran ventaja la qual es la gran velocidad con la que trabajan, y que esta velocidad esta muy por encima de cualquier framework.
